Modal Verbs

Practise B2 First transformations with modal verbs: certainty, possibility, impossibility, obligation, prohibition, advice, criticism, regret and ability.

Core patterns MUST / CAN’T MIGHT / MAY / COULD HAVE TO / NEEDN’T SHOULD / HAD BETTER BE ABLE TO / MANAGED TO

Study Mode

Review the main modal verb patterns before practising. In B2 First Part 4, the most important thing is to recognise the meaning behind the first sentence and express it with the correct modal structure in the second sentence.

1. Certainty, possibility and impossibility

  • Present certainty: must + base verb
  • Past certainty: must have + past participle
  • Present possibility: might / may / could + base verb
  • Past possibility: might / may / could have + past participle
  • Present impossibility: can’t + base verb
  • Past impossibility: can’t have + past participle

2. Obligation, prohibition and necessity

  • Present/future obligation: must / have to + base verb
  • Past obligation: had to + base verb
  • No obligation now: don’t have to / needn’t + base verb
  • Prohibition: mustn’t / can’t + base verb
  • Unnecessary action done: needn’t have + past participle
  • Unnecessary action not done: didn’t need to + base verb

3. Advice, criticism and regret

  • Advice now: should / ought to + base verb
  • Strong advice: had better + base verb
  • Regret about the past: should have + past participle
  • Criticism of a past action: shouldn’t have + past participle

4. Ability and achievement

  • Present ability: can / be able to + base verb
  • General past ability: could + base verb
  • Specific past achievement: was/were able to + base verb
  • Success after effort: managed to + base verb
  • Failure: couldn’t / wasn’t able to / didn’t manage to
